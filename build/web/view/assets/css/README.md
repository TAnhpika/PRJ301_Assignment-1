# Styles

This directory contains the CSS files for the project.
The project now uses vanilla CSS and does not require SCSS compilation.

## Files

- `dashboard.css`: Main layout styles for the dashboard (sidebar, header, grid).
- `dashboard-common.css`: Common utilities, variables, and component styles (cards, buttons, tables) used across all dashboards.
- `manager.css`: Specific styles for Manager role.
- `staff.css`: Specific styles for Staff role.
- `global-fonts.css`: Font definitions.
- `home.css`: Styles for the public landing page.

## Notes

- All CSS files are self-contained.
- Use `dashboard-common.css` for shared design tokens (colors, spacing).
